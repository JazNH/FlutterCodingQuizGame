# Python Quiz Game (Flutter)

A fun, bubbly quiz game focused on Python 3 concepts.

## Features
- Main menu with Play, Settings, High Scores
- Progressive difficulty: Easy -> Medium -> Hard
- Easy section ends randomly between question 7 and 30
- Medium section ends randomly between question 60 and 62
- Hard section continues through question 100
- 4 options per question
- Multi-answer support with clear prompts
- Game over on first wrong answer or timeout
- Run completion message when player clears question 100
- Name prompt before game starts
- High scores saved with timestamp
- Settings mode:
  - Quick: 30 seconds/question (default)
  - Study: 60 seconds/question

## Run
1. Install Flutter (compatible with your OS version).
2. In this folder run:
   - `flutter pub get`
   - `flutter run`
